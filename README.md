# 🎰 Royal Fortune Casino — Simulador de Roleta

Projeto acadêmico desenvolvido para a disciplina de **Ciência da Computação**.

## 🎯 Objetivo

Este simulador demonstra, na prática, como algoritmos de cassinos digitais são
programados para garantir a vantagem da banca — independentemente da estratégia
do jogador. O conceito conhecido como "house edge" é implementado de forma
intencional e transparente para fins educacionais.

## 🧠 Como funciona o algoritmo

Em uma roleta europeia justa, a casa tem ~2,7% de vantagem (1 número zero em 37).
Neste simulador, essa vantagem é elevada propositalmente para tornar o efeito
visível em poucas rodadas. O resultado de cada giro é calculado **antes** da
animação da roda — ou seja, a roleta gira bonita, mas o destino já está decidido.
